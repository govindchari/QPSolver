QOCO implements a primal-\/dual interior point method to solve second-\/order cone programs with quadratic objectives of the following form

\$\$ \textbackslash{}begin\{split\} \textbackslash{}underset\{x\}\{\textbackslash{}text\{minimize\}\} \textbackslash{}quad \& \textbackslash{}frac\{1\}\{2\}x$^\wedge$\textbackslash{}top P x + c$^\wedge$\textbackslash{}top x \textbackslash{} \textbackslash{}text\{subject to\} \textbackslash{}quad \& Gx \textbackslash{}preceq\+\_\+\textbackslash{}mathcal\{C\} h \textbackslash{} \textbackslash{}quad \& Ax = b \textbackslash{}end\{split\} \$\$

with optimization variable \$x \textbackslash{}in \textbackslash{}mathbb\{R\}$^\wedge$n\$ and problem data \$P = P$^\wedge$\textbackslash{}top \textbackslash{}succeq 0\$, \$c \textbackslash{}in \textbackslash{}mathbb\{R\}$^\wedge$n\$, \$G \textbackslash{}in \textbackslash{}mathbb\{R\}$^\wedge$\{m \textbackslash{}times n\}\$, \$h \textbackslash{}in \textbackslash{}mathbb\{R\}$^\wedge$m\$, \$A \textbackslash{}in \textbackslash{}mathbb\{R\}$^\wedge$\{p \textbackslash{}times n\}\$, \$b \textbackslash{}in \textbackslash{}mathbb\{R\}$^\wedge$p\$, and \$\textbackslash{}preceq\+\_\+\textbackslash{}mathcal\{C\}\$ is an inequality with respect to cone \$\textbackslash{}mathcal\{C\}\$, i.\+e. \$h -\/ Gx \textbackslash{}in \textbackslash{}mathcal\{C\}\$. Cone \$\textbackslash{}mathcal\{C\}\$ is the Cartesian product of the non-\/negative orthant and second-\/order cones, which can be expressed as

\$\$\textbackslash{}mathcal\{C\} = \textbackslash{}mathbb\{R\}$^\wedge$l\+\_\++ \textbackslash{}times \textbackslash{}mathcal\{Q\}$^\wedge$\{q\+\_\+1\}\+\_\+1 \textbackslash{}times \textbackslash{}ldots \textbackslash{}times \textbackslash{}mathcal\{Q\}$^\wedge$\{q\+\_\+N\}\+\_\+N\$\$

\$\$\textbackslash{}mathcal\{Q\}$^\wedge$\{q\+\_\+i\}\+\_\+i = \textbackslash{}\{(t,x) \textbackslash{}in \textbackslash{}mathbb\{R\} \textbackslash{}times \textbackslash{}mathbb\{R\}$^\wedge$\{q\+\_\+i -\/ 1\} \textbackslash{}; $\vert$ \textbackslash{}; $\vert$$\vert$x$\vert$$\vert$\+\_\+2 \textbackslash{}leq t \textbackslash{}\}\$\$\hypertarget{md__home_govind_Desktop_git_qoco_README_autotoc_md1}{}\doxysection{Bug reports}\label{md__home_govind_Desktop_git_qoco_README_autotoc_md1}
