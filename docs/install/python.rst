Python
==============

Pip
----
QOCO is on PyPI and can be installed as follows

.. code:: bash

   pip install qoco


Build from Source
-----------------

The Python wrapper for QOCO can also be built from source as follows 

.. code:: bash

   git clone https://github.com/qoco-org/qoco-python
   cd qoco-python
   pip install .
